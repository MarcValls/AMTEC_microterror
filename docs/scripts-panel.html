Arranque rapido del repo

Este panel resume los comandos principales del monorepo. El foco esta puesto en desarrollo local, sobre todo en levantar todo con un solo comando y en separar web y API cuando haga falta.

Monorepo con pnpm + turbo Script principal: pnpm dev Web directa en puerto 5173

Scripts principales

Usa estos comandos como punto de entrada. El primero es el que deberias disparar casi siempre para run dev.

Desarrollo completo

Levanta los paquetes con Turbo en paralelo. Es la opcion base para arrancar el entorno de desarrollo.

run dev
pnpm dev

Solo web

Arranca el frontend con el script local preparado para Vite.

frontend
pnpm dev:web

Solo API

Levanta la API local mediante el script bash del repo.

backend
pnpm dev:api

Web raw

Lanza directamente el paquete web con el filtro de pnpm, sin pasar por el script bash.

vite
pnpm dev:web:raw

Setup local

Prepara el entorno local antes de arrancar servicios por primera vez.

setup
pnpm setup:local

Utilidades

Scripts secundarios para validar, compilar o apagar lo que has levantado.

Build completa

pnpm build

Typecheck

pnpm typecheck

Checklist local

pnpm checklist:local

Parar entorno local

pnpm stop:local

Script auxiliar

Existe tambien un disparador extra del repo:

pnpm tu:flipao:inicia:app

Consejo practico: para trabajar normal, empieza por pnpm dev. Si estas tocando solo interfaz, usa pnpm dev:web.
